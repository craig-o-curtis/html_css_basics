Pseudo-classes

:hover, :visited, :link, :active, :focus

Pseudo-classes

  • :link - used for default link styling
  • :hover - changes styles when mouse hovers over element
  • :visited - changes styles after a link has been visited
  • :active - changes styles when clicked
  • :focus - changes styles when selected (with the Tab key) ** only works with a tags and button tags.
Anchor Button
Div Button

Notice how the a tag will keep the :visited styling after it is clicked on once, while the Div Button does not.

Specificity

User Agent > External CSS > Inline CSS

Importance is decided depending on the location of the CSS rule. Inline styles are the most powerful, yet the least commonly used.

It is best practice to keep your styles in an external stylesheet linked in the head of your HTML document.

element vs. element.class element

The more targeted the CSS rule, the more weight it has.

Example:

#specific_id { color: blue; }
.specificity_class { color: red; }
