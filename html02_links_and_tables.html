HTML Links and Tables

HTML Links - < a >

Use the < a > tag to make a link.

Add the href attribute and set it to the destination URL address.

< a href="http://www.google.com"> Google < /a >

The above code looks like:

Google

To make the browser open a new window, use the target keyword
and set the value equal to _blank.

< a href="http://www.google.com" target="_blank" > Google < /a >

Code output:

Google

Linked Image

Internal links

Internal < a > tags can reference parts on a webpage or website

Instead of using a URL, us a # hash tag to id of the location

This means your destination must have an id of the same name.

< a href="#top_of_page"> Go to top < /a >

Output:

Go to top

If you want to navigate to a parent directory, you can use the ../ to specify move up one level.

< a href="../"> Go to Parent Directory < /a >

Go to Parent Directory

Comments

Comments allow the human reader to undersand the code. It is good practice to explain complicated bits of your code to yourself or others.

<!-- This is an HTML comment -->

Comments are not shown in the browser.

Comments are also useful to debug or check your code. Add comments to a block of code for the browser to ignore it.
