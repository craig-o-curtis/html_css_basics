CSS Basics

CSS Selectors

selector { property: value; }
h1 { color: red; }

Generally just use an HTML tag name as the selector to create a new HTML rule.

Then specify the property and value. A complete list of properties and values can be found at Mozilla Developer's Network.

Colors

We can change the color of text using color keywords, HEXCODE, RGB or RGBA, and HSL or HSLA.

Backgrounds

You can use the background-color or background properties in CSS.
