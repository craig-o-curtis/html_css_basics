The Box Model

The Box Model

Elements have 4 parts:

  1. Actual content
  2. Padding
  3. Border
  4. Margin

Margin

The transparent area around the box that separates it from other elements.

Margin and Padding Values:

You may use px, em, rem, and percentage for values

You may also specify either one, two, three, or all four values for each side:

  1. One value covers all top, right, bottom, and left.

    .box_model_div {margin: 1em;}
    div
    .box_model_div {padding: 1em;}
    div

  2. The first value is for the top and bottom, and the second is for the right and left sides.

    .box_model_div {margin: 20px 10px;}
    div
    .box_model_div {padding: 20px 10px;}
    div

  3. The first value is for the top, the second value is for the right and left sides, and the third is for the bottom. .box_model_div {margin: 20px 10px 1px;}

    div
    .box_model_div {padding: 20px 10px 1px;}
    div

  4. Listing all four values, think of a clock starting from the top. The first value is the top, the second is the right, the third is the bottom, and the fourth is the left. .box_model_div {margin: 20px 10px 1px 15px;}

    div
    .box_model_div {padding: 20px 10px 1px 15px;}
    div

Specific Sides

We can also specify the side with the following:

Margin:

  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

Padding:

  • padding-top
  • padding-right
  • padding-bottom
  • padding-left

Border:

  • border-top
  • border-right
  • border-bottom
  • border-left

Position

position:

Types of Positioning:

  • static
      • .static_div {position: static;}
    • Normal positioning according to flow in document
    • Cannot set left, top, right, bottom, or z-index values
  • relative
      • .relative_div {position: relative; top: 10px; left: 20px;}
    • Can set left, top, right, bottom, and z-index values
    • Is the parent container of an position: absolute element
  • absolute
      • .absolute_div {position: absolute; top: 10px; margin-top: 10px; left: 20px; margin-left: 20px}
    • Simply, the child of a relatively-positioned element, creating a new canvas.
    • Taken out of document flow
    • Positioned at a certain position relative to its closest positioned ancestor or containing block.
  • fixed
      • .fixed_div {position: fixed;}

      Note:In this example, just look at the positioning in the document compared to the first teal-colored div. Due to this page using the flexbox positioning scheme, the fixed position is not visible.

    • Positioned to the screen's viewport
    • Doesn't move when scrolled
  • sticky
    • A hybrid of relative and fixed.
    • Note:Example does not work here because it is looking for the viewport top values.
